DOMKit - The Docs

Introduction

DOMKit is an open-source HTML elements and CSS styling library, written entirely in JavaScript, used to simplify the creation of HTML elements and CSS styling options.

In the current version (V1.0), DOMKit provides 3 main JS files: elements.js, displays.js and styles.js.

Adding DOMKit to Your Site

Download the src folder from the Github repository into your code, then add the following line above any <script> tags, in the <body> tag.

<script src="/path/to/DOMKit/src/[scriptName].js"></script>

Elements

This section outlines all functions used to create various HTML elements. These elements have some default styling applied to them, such as the navigation bar.

createDropdown(dropdownName, options, label, parent="body")

Creates and returns a dropdown element, and adds it to the top of the DOM.

dropdownName
The name of the dropdown element.

options
A list of strings, where each string represents an option for the dropdown menu. The values for these options will be each option converted to lowercase, for simplicity.

label
The string to label the dropdown menu.

parent
ID of the parent element where the table should be added to. Defaulted to 'body'.

Example

const cars = createDropdown("cars", ["Volkswagen", "Kia", "Mercedes-Benz"], "Select your favourite car brand.");

createForm(title, actionPage, inputData, parent="body")

Creates and returns a form element, and adds it to the DOM.

title
The title of the form element, that will become a <h1> tag.

actionPage
An external page where the data should be sent. Usually a PHP page.

inputData
An object containing input data for the form. Keys are the labels, and the values are the type of input.

parent
ID of the parent element where the table should be added to. Defaulted to 'body'.

Example

const signInForm = createForm("action.php", {"Username": "text", "Password": "password"});

createNavBar(links, bgColour, textColour="black", resetCSS=true)

Creates and returns a nav element, with default styling for a navigation bar, and adds it to the top of the DOM.

links
An object containing data about each link of the nav bar, keys being the name and values being the filepaths for said name.

bgColour
A list containing three values: red, green, blue. For the background colour of the navigation bar.

textColour
A string indicating what colour the text should be. Defaulted to "Black".

resetCSS
Indicates whether or not the cssReset() function should be called, which resets the browser's margin and padding values. Defaulted to "true".

Example

const myNav = createNavBar({'Home': '', 'About': ''}, [66, 106, 190]);

createTable(headers, rowData, hasBorders=true, parent="body")

Creates and returns a table element, and adds it to the DOM.

headers
A list containing the table headers.

rowData
A list of lists, where each list represents a row of the table.

hasBorders
Determines if the table should have borders. Defaulted to 'true'.

parent
ID of the parent element where the table should be added to. Defaulted to 'body'.

Example

const myTable = createTable(["header1", "header2"], [["item1-header1", "item1-header2"], ["item2-header1", "item2-header2"]]);

createVideoEl(filepath, controls=true, parentID="body")

Creates and returns a nav element, with default styling for a navigation bar, and adds it to the top of the DOM.

filepath
The filepath of the video.

controls
Determines whether or not the user can control the video by playing, pausing, etc. Defaulted to 'true'.

parentID
ID of the parent element where the table should be added to. Defaulted to 'body'.

Example

const websiteDesign1 = createVideoEl("media/web_design_1.mp3");

Displays

This section outlines each of the display functions, used to create neater layouts.

createFlexbox(flexDirection, textCentered=true, hideOverflow=true, elemCentered=true, parent="body")

Creates and returns a flexbox container.

flexDirection
The flex direction of the container. Either 'row' or 'column'.

textCentered
Determines if all text elements should be centered. Defaulted to 'true'.

hideOverflow
Determines if any overflowing elements should be hidden. Defaulted to 'true'.

elemCentered
Determines if all elements in the flexbox should be centered. Defaulted to 'true'.

parent
ID of the parent element where the table should be added to. Defaulted to 'body'.

Example

const myFlexbox = createFlexbox("column", textCentered=true, parent="flexboxWrapper");

createGrid(numBoxes, boxColour, gridTemplateArea, gap, width, height, parent="body")

Creates and returns a grid container.

numBoxes
The number of boxes that should be created inside the grid.

boxColour
A list containing three values: red, green, blue. For the background colour of each box element.

gridTemplateArea
A list, where each list inside the main list represents a row for the grid-template-area value. An `i`th box will have an ID of `box{i}`.

gap
Sets the distance between all box elements in the grid.

width
Width of each box that will be created and stored in the main DIV.

height
Height of each box that will be created and stored in the main DIV.

parent
ID of the parent element where the table should be added to. Defaulted to 'body'.

Example

const myGrid = createGrid(3, [0, 128, 0], [["box1", "box1", "box1", "box3"], ["box2", "box2", "box2", "box3"]], "10px", "100px", "100px", parent="gridWrapper");

Styles

This section outlines each of the styling functions, used to style HTML elements. All of these functions do not return the element, as the element's styling is changed directly.

centerEl(element)

Uses flexbox to center a given element.

element
The HTML element to be centered.

gradientBG(element, direction, gradientType, colours, textColour="black")

Adds a background gradient to an element.

element
The HTML element where the gradient will be applied, any type of HTML element will suffice.

direction
The direction of the gradient in `degrees`.

gradientType
Determines the gradient type to be used.

colours
A list of lists, where each list contains RGB values for each colour, allowing more than 2 colours.

textColour
Colour of the text inside the element. Defaulted to 'black'.

Example

gradientBG(element, "90deg", "linear", [[128, 0, 0], [0, 128, 0], [0, 0, 128]], "white");

textGradient(element, direction, colours)

Adds a gradient to an element's text.

element
The HTML element where the gradient will be applied, any type of HTML element will suffice.

direction
The direction of the gradient in `degrees`.

colours
A list of lists, where each list contains RGB values for each colour, allowing more than 2 colours.

Example

textGradient(element, "90deg", [[128, 0, 0], [0, 128, 0], [0, 0, 128]]);

shadowEffect(element, facing, spread, colour)

Adds a shadow effect to a given element.

element
The HTML element where the gradient will be applied, any type of HTML element will suffice.

facing
How far the shadow should spread in pixels.

spread
The direction that the element will face. Either `top-left`, `bottom-left`, `top-right`, `bottom-right`, or 'forward'.

colour
A list containing RGB values for the chosen colour.

Example

shadowEffect(element, "top-right", "20px", [128, 0, 0]);

glassmorphism(wrapperEl, section, bgImgPath, width="50%", padding="20px", addShadow=true, blur="5px")

Adds a glassmorphism effect to a given element.

wrapperEl
The HTML DIV element that includes the section element where the effect will be applied.

section
The section element within `wrapperEl`. This will include things like a header, text, a CTA button, etc.

bgImgPath
The path to the image that will be placed on the background.

width
The width of the element with ID `sectionID`. Defaulted to '50%', half of the width of its parent element.

padding
Determines the padding that'll get applied to all elements in the element with ID `sectionID`. Defaulted to '20px'.

addShadow
Determines if the glassmorphism effect should have a shadow effect. Defaulted to 'true'.

blur
The amount of blur in the backdrop filter. Defaulted to '5px'.

Example

glassmorphism(divEl, mySection, "media/cool-bg.jpg");
